Audio Analysis Accuracy: Manual vs Program

This page documents two manual test comparisons between professional audio tools (Voxengo Span, Waves PAZ, Waves InPhase) and the automated analysis metrics generated by the ZoundZcope backend. The goal is to transparently demonstrate how closely the program's audio analysis matches manual measurements and expert listening impressions.

Note: Manual testing usually analyzes selected sections or brief moments of a track, whereas the automated program analyzes the entire track from start to finish. This difference in scope can cause metric variations such as RMS, peak levels, and spectral characteristics. Plugin sensitivities and measurement methods may also vary, contributing to discrepancies.
